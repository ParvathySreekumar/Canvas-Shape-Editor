2D Shape Editor Help

Toolbar Buttons

New

Start a new diagram. This will clear the canvas and any unsaved changes will be lost.

Undo

Undo the last action. This will revert the canvas to its state before the last action.

Redo

Redo the last undone action. This will reapply the last action that was undone.

Copy

Copy the selected shape. The copied shape can be pasted using the Paste button.

Paste

Paste the copied shape onto the canvas. The pasted shape will be placed at the current cursor position.

Save JSON

Save the diagram as a JSON file. This will allow you to reload the diagram later using the Load button.

Save JPEG

Save the diagram as a JPEG image. This will create an image file of the current diagram.

Grid

Toggle the visibility of the grid on the canvas. This can help with aligning shapes.
